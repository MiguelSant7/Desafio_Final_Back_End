API de Alunos e Cursos

Bem-vindo! Esta é uma API simples para gerenciar alunos, cursos e matrículas.

Endpoints principais

  • GET /alunos — listar alunos
  • POST /alunos — criar aluno
  • GET /cursos — listar cursos
  • POST /cursos — criar curso
  • POST /matriculas — criar matrícula

Regras

  • Matrícula única por aluno+curso
  • Máx. 5 matrículas ativas por aluno
  • Validações de campos obrigatórios

Exemplos rápidos (curl)

curl -X POST http://{host}/alunos -H 'Content-Type: application/json' -d '{"nome":"Ana","email":"ana@ex.com"}'
curl -X POST http://{host}/cursos -H 'Content-Type: application/json' -d '{"titulo":"Matematica"}'
curl -X POST http://{host}/matriculas -H 'Content-Type: application/json' -d '{"aluno_id":1,"curso_id":1}'

Dicas

Se você implantou na Render, substitua {host} pelo domínio que o Render forneceu (ex.: meu-app.onrender.com).
